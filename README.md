# 🧾 BillSmart

> AI-powered receipt scanner and expense tracker built for Indian users.

🔗 **[Live Demo](https://billsmart-8aqppmtesqkrqcxfk5mpyn.streamlit.app/)**

---

## 📌 The Problem

500 million+ Indians use UPI daily but have no simple way to track their spending. Most expense apps are complex, manual, or not built for Indian receipts.

## 💡 The Solution

BillSmart lets you photograph any receipt and instantly get:
- Itemized expense breakdown
- Automatic category detection
- Spending history and charts
- All in one clean dashboard

---

## ✨ Features

- 📸 **Receipt Scanning** — Upload any receipt photo
- 🔍 **OCR Processing** — Extracts text using EasyOCR
- 💰 **Amount Detection** — Finds all prices automatically
- 🏷️ **Smart Categorization** — Food, Grocery, Medical, Transport
- 📊 **Dashboard** — Visual spending charts
- 💾 **History Tracking** — Save and review past expenses

---

| Technology | Purpose |
